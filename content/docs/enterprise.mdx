面向已经把 AI 调用接入到产品或内部流程的团队，我们提供独立于个人套餐的企业方案。**核心差异不是更多的 token，而是稳定性、响应速度与合规支持**。

## 我们解决的问题

如果你的团队遇到过以下任何一种情况，企业方案可能比个人套餐更适合你：

- **波动影响生产**：模型调用偶发失败，需要自己在代码里写 fallback、重试、监控
- **额度结构不灵活**：日卡 / 月卡难以支撑团队多人多场景的并发需求
- **支持响应慢**：出问题时联系客服等好几个小时，业务停摆
- **缺合规材料**：财务要发票、法务要合同、安全团队要数据处理协议
- **没有可见性**：不知道每个项目花了多少 token、谁在跑什么、成本怎么分摊

## 我们不卖什么

为了不浪费你的时间，以下需求**不在我们当前的服务范围内**：

- 用于训练自有大模型的批量数据请求（请直接联系模型提供方）
- 涉政、涉黄、涉赌、涉诈骗内容的 API 调用
- 任何要求规避模型提供方使用条款的方案

</Callout>

<Callout type="info" title="可以按年付费吗？是否有折扣？">

支持按年付费，提供 8.5 折优惠（仅适用于企业旗舰版及以下）。企业定制版按合同约定。

可以。一个 API key 即可访问全部模型，按实际调用模型计费。

</Callout>

<Callout type="info" title="如果用量超过套餐怎么办？">

默认行为是暂停服务并发邮件提醒。你可以选择：(a) 升级到更高档；(b) 临时增量购买；(c) 在合同中约定按用量结算（仅企业定制版）。

</Callout>

<Callout type="info" title="支持私有化部署吗？">

仅企业定制版支持。私有化部署形态：网关本体部署在你的 VPC，模型调用走我们维护的合规算力池，请求内容不经过共享通道。具体方案请联系商务。

</Callout>
