faq:
  - q: 灵眸 AI 是什么？
    a: 灵眸 AI 是稳定、靠谱的 Claude / GPT / Gemini API 中转服务（俗称"中转站"），通过一个 API 密钥即可在 Claude Code、Codex CLI、Cursor、VS Code、OpenCode、Cherry Studio 等主流 AI 编程工具中使用 Anthropic / OpenAI / Google 与国产大模型。
  - q: 灵眸 AI 支持哪些工具？
    a: 支持 Claude Code CLI / 桌面版 / VS Code 插件、Codex CLI（Mac / Windows / 服务器）/ Codex 桌面版、Cursor、Cherry Studio、OpenCode、Hermes Agent、IDEA Kilo Code、CC Switch 等。
  - q: 国内能直接使用吗？
    a: 可以。接入网关部署在中国国内，国内网络可直接调用，无需自建代理或科学上网。
