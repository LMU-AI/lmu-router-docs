邀请好友注册灵眸，对方每次充值或订阅，你都能持续获得佣金——终身绑定，长期收益。

---

## 规则说明

- 新用户通过你的邀请链接注册，即与你**终身绑定**
- 他们每次充值或订阅，你获得 **10% 账面佣金**
- 佣金随被邀请人实际消费逐步释放（充值按余额消费进度，订阅按生效天数）
- 可使用佣金可**直接转入账户余额**用于 API 消费，或申请提现至微信 / 支付宝 / 银行卡
- 新人首次付费额外赠金 **¥2.00**

---

## 如何参与

登录后台，左侧菜单点击「**推荐有奖**」，即可看到你专属的邀请链接和邀请码。

### 方式一：分享邀请链接

复制你的专属链接发给朋友，对方点击后注册即完成绑定，格式如下：

```
https://api.lmuai.com/register?ref=你的邀请码
```

### 方式二：分享邀请码

将你的邀请码告知对方，对方注册时手动填入即可完成绑定。

---

## 佣金说明

| 项目 | 说明 |
|------|------|
| 返佣比例 | **10%**（被邀请人每笔充值 / 订阅） |
| 新人赠金 | 首次付费赠 **¥2.00** |
| 绑定方式 | 通过链接注册即终身绑定 |
| 佣金释放 | 按被邀请人实际消费进度逐步释放 |
| 佣金使用 | 转入余额（即时可用）或申请提现 |

---

## 查看与提现

登录后台后，左侧菜单点击「**推荐有奖**」可查看：

- 累计佣金、可使用佣金、已释放金额
- 已邀请人数及每笔返佣明细
- 一键将可用佣金转入余额，或申请提现

<Callout type="info" title="小提示">

邀请链接可隐藏推广痕迹，分享更自然。登录后台「推荐有奖」页面可下载专属二维码，方便在社群、朋友圈等场景传播。

</Callout>
