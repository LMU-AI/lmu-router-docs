下方为灵眸 API 当前支持的所有模型名称，**点击任意卡片即可一键复制模型 ID**，可直接粘贴到 `settings.json`、`config.toml` 或工具的模型选择框中。

<Callout type="info" title="如何使用模型名">

将复制到的模型 ID 填入对应工具的配置项即可：

- **Claude Code** → `settings.json` 中的 `"model"` 字段
- **Codex CLI** → `~/.codex/config.toml` 中的 `model = "..."`
- **Cherry Studio / VS Code 插件** → 模型选择下拉框中粘贴

</Callout>
