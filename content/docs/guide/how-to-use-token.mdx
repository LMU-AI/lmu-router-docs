## Token 类型

| 类型 | 说明 |
|------|------|
| 个人 Token | 绑定到你的账号，使用你的额度 |
| 渠道 Token | 管理员分配，独立额度管理 |

## 查看用量

进入 **控制台 → 用量统计**，可查看：

- 按日/月的调用次数
- Token 消耗量
- 费用明细
- 各模型使用占比

## 额度限制

创建 Token 时可设置以下限制：

- **总额度**：该 Token 可消耗的最大额度（留空表示不限）
- **每分钟请求数**：限流保护
- **模型白名单**：限制该 Token 只能调用指定模型

## 重置与撤销

- **重置额度**：在 Token 管理页可手动重置已用额度
- **撤销 Token**：删除 Token 后立即失效，已发出的请求不受影响
