## 第一步：注册账号

访问注册地址：[https://api.lmuai.com/register?ref=vJaWWr4T](https://api.lmuai.com/register?ref=vJaWWr4T)

输入邮箱和密码完成注册。

## 第二步：订阅套餐或充值

登录后，在左侧菜单栏点击「**充值/订阅**」：

- **订阅套餐**：选择适合自己的订阅计划，按月享用固定额度
- **直接充值**：按需充值余额，灵活使用各类模型

完成支付后即可使用。

## 第三步：创建 API 密钥

1. 点击左侧菜单栏「**API 密钥**」
2. 点击「创建密钥」，名称随意填写
3. 创建完成后，**请记住你的 API 密钥**（即 `sk-` 开头的密钥）

## 第四步：配置使用

**API 请求地址：**

根据你使用的工具或模型，选择对应协议的请求地址：

**Anthropic 协议**（适用于 Claude Code、Cherry Studio、Trae、Kilo Code 接 Claude / 国产模型等）：

```
https://api.lmuai.com
```

**OpenAI 协议**（适用于 Codex CLI、Codex App、VS Code/Cursor 接 GPT、Kilo Code 接 OpenAI 模型等）：

```
https://api.lmuai.com/v1
```

<Callout type="warn" title="两者区别（最容易填错的地方）">

- **Anthropic 协议** 不带 `/v1` 后缀
- **OpenAI 协议** 必须带 `/v1` 后缀

填错协议会导致 401 / 404 报错。详细协议说明、SDK / curl 示例和 Claude Max 分组限制见 **[接入协议 →](./api-protocols)**。

</Callout>

记住这几样东西，后续所有工具配置都需要用到：
- **API Key**：你在后台生成的 `sk-` 开头的密钥（两种协议通用）
- **Anthropic 协议地址**：`https://api.lmuai.com`
- **OpenAI 协议地址**：`https://api.lmuai.com/v1`
