## 第一步：注册账号

访问注册地址：[https://clawapi.fulitimes.com/register?ref=vJaWWr4T](https://clawapi.fulitimes.com/register?ref=vJaWWr4T)

输入邮箱和密码完成注册。

## 第二步：兑换套餐

1. 登录后，在左侧菜单栏点击「**兑换**」
2. 输入购买时自动发货订单上的**兑换码**，完成套餐兑换
3. 点击「**我的订阅**」可查看当前已兑换的套餐和剩余额度

## 第三步：创建 API 密钥

1. 点击左侧菜单栏「**API 密钥**」
2. 点击「创建密钥」，名称随意填写
3. **分组**选择你兑换码对应的套餐订阅，然后创建
4. 创建完成后，**请记住你的 API 密钥**（即 `sk-` 开头的密钥）

> 在 API 密钥列表中，可双击切换分组来使用不同的套餐

## 第四步：配置使用

**API 请求地址：**

```
https://clawapi.fulitimes.com
```

记住这两样东西，后续所有工具配置都需要用到：
- **API Key**：你在后台生成的 `sk-` 开头的密钥
- **API 地址**：`https://clawapi.fulitimes.com`
