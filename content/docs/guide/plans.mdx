套餐额度**每日刷新**，目前有以下规格：

| 套餐 | 每日额度 | 说明 |
|------|----------|------|
| 标准版 | 90刀/天 | 适合日常使用 |
| 进阶版 | 135刀/天 | 适合中度使用 |
| 专业版 | 200刀/天 | 适合重度使用 |
| 大额套餐 | 400刀、800刀、2000刀 | 联系客服充值 |

## 套餐周期

- **天卡**：兑换后 24 小时内有效
- **周卡**：7天有效期
- **月卡**：30天有效期

## 续期 vs 增量

<Callout type="warn" title="重要区别">

- **相同套餐** = 续期（延长时间，额度不叠加）
- **不同套餐** = 增量（额度叠加，互不影响）

</Callout>

**示例：**

- 你购买了「90刀天卡」后，再购买「90刀天卡」→ 总时长变成 **2天**（续期）
- 你购买了「90刀月卡」后，再购买「90刀天卡」→ 月卡额度用完后可切换天卡增量使用（增量）

**增量使用方法：** 在后台 API 密钥页面，双击当前密钥切换到对应套餐的分组即可。

<Callout type="info" title="注意">

不要重复购买相同套餐用于增量！如需增量请购买不同套餐。

例如：你是天卡用户，如果再拍天卡会变成续期而非增量，应选择拍月卡或重新注册账号。

</Callout>

## 可用模型

目前支持以下模型：

### GPT-5.1 系列
- `gpt-5.1`
- `gpt-5.1-codex`
- `gpt-5.1-codex-mini`
- `gpt-5.1-codex-max`

### GPT-5.2 系列
- `gpt-5.2`
- `gpt-5.2-codex`

### GPT-5.3 系列
- `gpt-5.3-codex`

### GPT-5.4
- `gpt-5.4`

<Callout type="info" title="关于 GPT-5.3/5.4 xhigh">

目前已支持 CODEX 5.4 xhigh 系列，但由于尚未全面推送，很多工具无法在下拉列表中直接选择该模型。

工具配置好后**默认使用 gpt-5.3-codex xhigh**，修改后也无法自由切换模型。

如不小心切换到其他模型，可修改 `.codex` 文件夹下的 `config.toml`，将 `model` 改回 `gpt-5.4`，重启终端或 VS Code 即可恢复。

</Callout>
