**Claude Code for Desktop** 是 Anthropic 官方推出的桌面客户端，新版本起支持自定义第三方推理端点，可直接接入灵眸 API。无需手动编辑 `settings.json`，全程在 GUI 中完成。

<Callout type="info" title="哪个适合我？">

- 想在终端使用 → [Claude Code CLI](./claude-code)
- 想在 VS Code 内使用 → [VS Code 插件](./claude-code-vscode)
- **想用独立桌面 App** → 本页

三者使用同一把灵眸 API Key，按习惯选择即可。

</Callout>

<Callout type="warn" title="桌面版需要可访问外网的网络环境">

与 CLI 不同，**Claude Code 桌面客户端本身需要能正常访问海外网络**：下载页和客户端的登录、激活、版本校验等流程都会直连 Anthropic 官方服务器。

因此即便你已经把 Base URL 指向灵眸，仍需开启**可访问外网的网络代理（俗称「魔法」/ 科学上网）**，否则会卡在下载或启动阶段。配置完灵眸后，模型对话本身走的是灵眸节点，但客户端的基础联网能力依旧依赖外网代理。

</Callout>

## 第一步：下载最新版本

<Callout type="warn">

**必须使用最新版本**，旧版桌面客户端不支持自定义第三方端点（即没有 `Configure Third-Party Inference` 选项）。

</Callout>

前往 [Claude 官网下载页](https://claude.ai/download) 下载并安装对应平台的客户端，安装完成后启动应用。

> 下载页位于海外，**需先开启外网代理（魔法）** 才能正常打开和下载安装包。

---

## 第二步：开启 Developer Mode

第三方端点配置入口默认隐藏，需要先打开开发者模式。

1. 顶部菜单栏点击 **Help**
2. 选择 **Troubleshooting**
3. 点击 **Enable Developer Mode**

开启成功后，**顶部菜单栏会新增一项 `Developer` 菜单**。

---

## 第三步：配置灵眸 API

1. 点击顶部菜单栏 **Developer**
2. 选择 **`Configure Third-Party Inference …`**
3. 在弹出的配置面板（**Connection → Gateway**）中填写：

| 字段 | 取值 |
|------|------|
| **Credential kind** | `Static API key` |
| **Gateway base URL** | `https://api.lmuai.com` |
| **Gateway API key** | 灵眸后台生成的 `sk-` 开头密钥 |
| **Gateway auth scheme** | `bearer` |

![Claude Code 桌面版 Configure third-party inference 面板填写灵眸 Gateway base URL 与 API key](/images/tools/claude-desktop/claude-code-desktop.png)

4. 点击右下角 **`Apply locally`** 保存，随后**重启客户端**生效

<Callout type="info">

Base URL 填写 **`https://api.lmuai.com`**（不带 `/v1` 后缀），桌面版与 CLI 走的都是 Anthropic 协议。

</Callout>

<Callout type="warn">

配置面板出现前，客户端需要先完成联网登录/激活，这一步**必须开启外网代理（魔法）**。建议在打开客户端、进行配置的全过程都保持代理可用。

</Callout>

---

## 验证配置

打开任意会话向 Claude 发送一句话，若能正常返回回复即说明配置成功。如果想确认请求确实走了灵眸：

- 灵眸后台 **API 密钥 → 用量统计** 页面会即时记录到本次请求
- 客户端右下角的状态指示通常会显示当前模型

---

## 切换模型 / 使用国产模型

如要使用国产模型（Qwen、DeepSeek、GLM、Kimi 等），在 `Configure Third-Party Inference` 的 **Model** 字段填入对应模型 ID 即可，例如：

- `qwen3.6-plus`
- `deepseek-v3.2`
- `glm-5.1`

完整可用模型清单见 [模型广场](../guide/models)，可一键复制 ID。

## 常见问题

### 找不到 `Configure Third-Party Inference` 选项

桌面版本太旧，前往官网下载最新版重新安装即可。该选项是较新版本才加入的功能。

### Developer 菜单没有出现

确认 `Help → Troubleshooting → Enable Developer Mode` 已勾选；若仍未出现，**完全关闭并重启客户端**（Mac 上需要在 Dock 右键 Quit，不只是关闭窗口）。

### 提示 401 / Unauthorized

- 检查 API Key 是否复制完整（`sk-` 开头）
- 确认 Base URL 是 `https://api.lmuai.com`，**不要带 `/v1`**
- 灵眸后台确认密钥所属分组有可用额度

### 卡在下载 / 登录 / 启动界面

桌面客户端的下载与基础联网（登录、激活、版本校验）都需要直连海外服务器。请确认**外网代理（魔法）已开启且工作正常**，再重新打开客户端。这一限制与是否配置灵眸 Base URL 无关。

### 想恢复使用 Anthropic 官方接口

`Developer → Configure Third-Party Inference` 中清空配置或选择默认即可恢复。
