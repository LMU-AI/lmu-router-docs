## 安装插件

1. 打开 IntelliJ IDEA，进入 **Settings → Plugins → Marketplace**
2. 搜索 **Kilo Code**，找到插件后点击安装
3. 安装完成后重启 IDEA

---

## 配置插件

打开 Kilo Code 设置面板，按照你要使用的模型类型选择对应的方案：

<Tabs items={['国产模型 / Claude（Anthropic）', 'OpenAI 模型（OpenAI Compatible）']}>
<Tab value="国产模型 / Claude（Anthropic）">

适用于 `claude-opus-4-7`、`qwen3.6-plus`、`deepseek-v3.2`、`glm-5.1`、`kimi-k2.6` 等所有 Claude 与国产模型。

| 字段 | 取值 |
|------|------|
| **API Provider** | `Anthropic` |
| **Anthropic API Key** | 灵眸后台生成的 `sk-` 开头密钥 |
| **Use custom base URL** | ✅ 勾选 |
| **Base URL** | `https://clawapi.fulitimes.com` |
| **Model** | 例如 `claude-opus-4-6`、`qwen3.6-plus` |

![Kilo Code 选择 Anthropic 提供商，勾选 Use custom base URL 并填入 https://clawapi.fulitimes.com，模型选择 claude-opus-4-6](/images/tools/Kilo/kilo.png)

<Callout type="info">

需要勾选 **Use custom base URL**，否则插件会请求 Anthropic 官方接口而非灵眸中转。

</Callout>

</Tab>
<Tab value="OpenAI 模型（OpenAI Compatible）">

适用于 `gpt-5.5`、`gpt-5.4`、`gpt-5.3-codex`、`gpt-5.2` 等所有 OpenAI 模型。

| 字段 | 取值 |
|------|------|
| **API Provider** | `OpenAI Compatible (Responses)` |
| **Base URL** | `https://clawapi.fulitimes.com/v1` |
| **API Key** | 灵眸后台生成的 `sk-` 开头密钥 |
| **Model** | 例如 `gpt-5.5` |

![Kilo Code 选择 OpenAI Compatible (Responses) 提供商，Base URL 为 https://clawapi.fulitimes.com/v1，模型设置为 gpt-5.5](/images/tools/Kilo/kilo-openai.png)

<Callout type="warn">

OpenAI 模型的 Base URL **必须以 `/v1` 结尾**，与 Anthropic 协议（不带 `/v1`）不同，请勿混用。

</Callout>

</Tab>
</Tabs>

## 注意事项

- 使用 **Claude / 国产模型** → 选 `Anthropic` 协议，Base URL 为 `https://clawapi.fulitimes.com`（不带 `/v1`）
- 使用 **OpenAI 模型** → 选 `OpenAI Compatible (Responses)` 协议，Base URL 为 `https://clawapi.fulitimes.com/v1`（带 `/v1`）
- API Key 在两种协议下都填写灵眸后台生成的 `sk-` 开头密钥
- 配置完成后点击右上角 **Save** 保存设置即可使用
